# Exercício 1 — Função ou Procedimento?

1. **`Str()`**
   - **Classificação:** Função
   - **Justificativa:** Converte e retorna um valor numérico formatado como string (texto).

2. **`QOut()`**
   - **Classificação:** Procedimento
   - **Justificativa:** Apenas executa a ação de exibir uma expressão ou texto no terminal e pular linha, sem retornar um valor.

3. **`Date()`**
   - **Classificação:** Função
   - **Justificativa:** Retorna um valor do tipo data contendo a data atual do sistema.

4. **`Len()`**
   - **Classificação:** Função
   - **Justificativa:** Retorna um número inteiro correspondente ao tamanho de uma string ou ao número de elementos de um array.

5. **`Upper()`**
   - **Classificação:** Função
   - **Justificativa:** Recebe uma string e retorna uma nova string com todos os caracteres alfabéticos convertidos para maiúsculas.

6. **`ClearScreen()`**
   - **Classificação:** Procedimento
   - **Justificativa:** Executa apenas a ação de limpar a tela do console, sem retornar nenhum valor.
